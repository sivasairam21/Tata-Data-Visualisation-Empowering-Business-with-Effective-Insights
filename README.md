# Tata-Data-Visualisation-Empowering-Business-with-Effective-Insights

This project focuses on creating an interactive **Power BI dashboard**
to empower business decision-making through effective data visualization.

The dashboard provides insights into sales performance, customer behavior,
and geographical distribution to support strategic planning.
